Animated Calendar Planner

A modern, interactive calendar web app built with **React + Tailwind CSS**.

Features

* Monthly calendar view
* Date range selection (start → end)
* Month-wise notes (each month stores its own notes)
* Dynamic themes (color + image per month)
* Smooth UI transitions
* Responsive design
* Weekend highlighting
* Today indicator

Key Concepts Used

* React state management
* Controlled components
* Conditional rendering
* Tailwind CSS styling
* Date handling with `date-fns`
* Dynamic theming
* Layout stability (fixed grid)
